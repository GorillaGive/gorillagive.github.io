Create An Event

Event hosts can easily create events once registered on {{ site.company }}. Once an event is created, mantinence of the event is available online as well.

Invite Guests

Use {{ site.company }} to invite your guests through social media such as Facebook, Twitter, LinkedIn or Google Play. Invites can also be sent out in an email format.

Layout An Event

{{ site.company }} lets event hosts use an interface to layout the event floorplan so that once the event comes, setup will have a layout to follow.

Table Layout

Once event hosts input the size of the hall where the event is taking place, they can drag and drop tables and stage to have a floorplan all set for event night.

Guest Check-In

{{ site.company }} has also added a feature for organization hosts to have guest check-in setup through a mobile app. This saves costs on how many people are required at guest check ins.

Auction Software

{{ site.company }} offers auction software that gives the event host an easy way to track items being auctioned off. There is also the option to host silent auction through mobile and the web during an event. Guests and users are also able to directly donate to organizations through a mobile app, useful if they cannot attend an event.

Tax Automation through Intuit TurboTax

{{ site.company }} has partnered with Intuit TurboTax to give guests and organizations an easy way to keep track of their donations.
